# PrepPilot AI Day 1 Project Log

## Product Direction

PrepPilot AI is a web application that helps a student prepare for a job interview. The user enters a job description or posting link and the interview date. The app creates a time-aware study plan with tests, coding practice, mock interviews, and final revision before the interview.

## Today’s Goal

Create the first working foundation:

- Project structure
- FastAPI backend skeleton
- First prep-plan API
- Deterministic planning logic
- Tests for the planner
- Word document log for the project

## MVP Features

- Paste a job description
- Enter interview date and available prep hours
- Detect likely skills from the job description
- Generate a day-by-day prep plan
- Include diagnostic, exams, coding practice, mock interview, and revision tasks

## Future Features

- OpenAI-powered job description analysis
- AI-generated exams and answer feedback
- Conversational mock interview mode
- Coding challenge runner
- User accounts and progress dashboard
- Interview experience database that improves future preparation
